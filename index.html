은행 선택

거래내역을 다운로드한 은행 또는 서비스를 선택하세요.
그랜터 파일 특징
· 금액 양수 → 입금 / 음수 → 출금 자동 분류
· 계정과목 컬럼 → 자동 import
· 거래수단 컬럼 → 계좌별 자동 분류 (한 파일에 여러 계좌 OK)
컬럼이 자동 감지됐어요.
